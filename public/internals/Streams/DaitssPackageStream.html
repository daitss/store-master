Overview

This data stream returns key/value pairs of fixity information as known by DAITSS. The keys are sorted, unique URLs (strings) showing the store-master provided location. The values are a datamapper-provided struct giving the expected MD5, SHA1 and Size of the package, as well as its IEID.

def read 
  return if eos?

  if @buff.empty?
    @buff = Daitss::Package.package_copies @ieids.shift(CHUNK_SIZE)
  end

  return if @buff.empty?

  datum = @buff.shift
  return datum.url, datum
end
