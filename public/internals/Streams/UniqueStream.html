Overview

UniqueStream takes a stream and filters it so that the returned stream’s keys are always unique; if a UniqueStream encounters two identical keys, it returns the key/value pair of the first of them, discarding the second.

It supports unget

def get
  return if eos?

  if @ungot
     @ungot = false
     return @unbuff[0], @unbuff[1]
  end

  ku, vu = @stream.get

  loop do
    break if eos?
    k, v = @stream.get
    if k != ku
      @stream.unget
      break
    end
  end

  @unbuff = [ ku, vu ]
  return ku, vu
end
