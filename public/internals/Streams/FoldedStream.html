Overview

FoldedStream is initialized from stream, returning a stream that has folded the values for identical keys together in an array. Thus the values for a FoldedStream are always an array, possibly of mixed arity. As for all streams, the keys must be sorted.

def read
  return unless upcoming = @stream.get
  vals = [ upcoming[1] ]

  loop do
    next_record = @stream.get

    if next_record.nil?
      return upcoming[0], vals
    end

    if next_record[0] == upcoming[0]
      vals.push next_record[1]
    else
      @stream.unget
      return upcoming[0], vals
    end
  end
end
