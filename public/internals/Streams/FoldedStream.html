Overview

FoldedStream is a stream filter; given a stream, it returns a stream that has folded values for identical keys together in an array. Thus the values for a FoldedStream are of mixed arity, but will always be an array. As for all streams, the keys must be sorted.

It subclasses UniqueStream and supports one level of unget

def get
  return if eos?

  if @ungot
     @ungot = false
     return @unbuff[0], @unbuff[1]
  end

  ku, vu = @stream.get

  @vals = Array.new
  @vals << vu

  loop do
    break if eos?
    k, v = @stream.get
    if k != ku
      @stream.unget
      break
    else
      @vals << v
    end
  end

  @unbuff = [ ku, @vals ]
  return ku, @vals
end
